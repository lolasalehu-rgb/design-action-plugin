## Tweet 2: The Problem

As a product designer, my evidence lives in:

→ PM syncs (Granola, Otter, Fireflies)
→ Sprint boards (Jira, Linear, GitHub)
→ Stakeholder threads (Slack, Discord)
→ Design reviews (Figma comments)
→ User interviews (video recordings)

Across 2 streams and 6 meetings/week.
I was losing decisions between meetings.

---

## Tweet 3: The Solution

design-action is a Claude Code plugin that:

→ Gathers evidence from all your tools
→ Synthesizes pain points, decisions, and quotes
→ Suggests the right artifact type
→ Creates it with full source citations

Every design element traces back to where it was discussed.

---

## Tweet 4: Install + Try

```
npx skills add lolasalehu-rgb/design-action-plugin
```

Then run `/setup` to connect your tools.

Works with Granola, Otter, Fireflies, Jira, Linear, GitHub Issues, Slack, Figma, Penpot — and more.

Also works with Cursor, Copilot, and Cline via skills.sh.
